SciPy 2018 Altair Tutorial

Link to GitHub Repository

Exploring a new dataset visually can provide quick intuition into relationships within the data. There are a few well-developed visualization packages in Python, but they often have very imperative APIs that force the user to focus on the mechanics of the visualization (tick locations, axis limits, legends, etc.), rather than the salient relationships within the data.

This tutorial will introduce data visualization with Altair, a package designed for exploratory visualization in Python that features a declarative API, allowing data scientists to focus more on the data than the incidental details. Altair is based on the Vega and Vega-Lite visualization grammars, and thus automatically incorporates best practices drawn from recent research in effective data visualization.

The tutorial will provide an introduction to the Altair package and its API, and will also dive into the core concepts of effective data visualization that can be applied using any visualization package or tool.

Audience:

This tutorial is for data scientists and others who work with data in Python, who wish to more effectively and easily explore datasets visually.

This tutorial assumes attendees are familiar with basic data manipulation using the Pandas package.

By the end of this tutorial, attendees will be familiar with visualizing data in Altair, and its approach to thinking about visualization.
